## 社团简介
民商法协会成立于2008年，由本院蔡元庆和王茂祺老师指导，是融合了学术研究和社会实践为一体的学生社团。以“深究法学理论，扩展学术实践，培养法律思维，营造学术氛围”为宗旨，组织展开案例讨论会，经验讲座等各类活动，以此研究学习各类经典的民商热点问题。
