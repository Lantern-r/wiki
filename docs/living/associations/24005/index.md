## 社团简介
深圳大学历史研习社是人文学院历史系下由学生自主创办的学术型社团。旨在自下而上地在历史系师生之间搭建起课外的交流平台，聚集各专业、各年级中对历史学感兴趣的同学开展学术活动。社团现下设口述史研究部、公众史学研究部、深港澳区域社会史研究部、社务部四个部门，共有36名同学加入。日常学术活动包括开办全社及各部门的小范围读书会、史料研读活动，承担人文学院历史系讲座宣传和会议纪要撰写工作。
