## 社团简介
深圳大学排球协会（即“深大排球俱乐部”）自2006年创建以来，始终致力于推广校园排球运动，搭建“以球会友”交流平台。协会在校体育部、体育联盟指导下开展工作，重心在校园排球普及、竞赛，广泛动员校内排球爱好者、各学院院队投身排球运动，促进基层排球发展。
